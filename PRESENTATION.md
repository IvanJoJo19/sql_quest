# Сценарий видеопрезентации

Длительность: 2-3 минуты.

## 1. Проблема

Новичкам сложно изучать SQL только по теории. Нужно место, где можно сразу писать запросы, видеть результат, получать проверку и подсказки.

## 2. Решение

SQL Quest - это интерактивный тренажер SQL. Пользователь выбирает задание, пишет запрос, запускает его на учебном датасете и получает обратную связь.

## 3. Демонстрация MVP

Показать главный экран:

- слева список заданий и прогресс;
- справа описание текущей задачи;
- ниже SQL-редактор;
- после запуска появляется таблица результата.

Показать сценарий:

1. Открыть задание `Выбрать студентов`.
2. Запустить готовый запрос.
3. Показать сообщение, что задание решено.
4. Перейти к заданию `Фильтр по городу`.
5. Специально изменить запрос неправильно и показать ошибку проверки.
6. Нажать `Подсказка ИИ`.
7. Исправить запрос и показать успешный результат.
8. Перейти к заданию `Студенты и прогресс`, чтобы показать `JOIN`.
9. Перейти к заданию `Сколько студентов в городах`, чтобы показать `GROUP BY`.

## 4. Ценность

Проект помогает учить SQL через практику. Пользователь не просто читает теорию, а сразу применяет ее, получает результат и подсказки.

## 5. Что дальше

В полноценной версии запросы будут выполняться на backend в изолированной PostgreSQL-песочнице. ИИ-помощник будет подключен через LLM API и сможет объяснять ошибки пользователя более гибко.
