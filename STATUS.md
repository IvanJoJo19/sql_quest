# Статус проекта

## Что уже сделано

- Выбрана тема: геймифицированный SQL-тренажер с ИИ-помощником.
- Подготовлено описание проекта в `README.md`.
- Описаны ценность, фичи MVP, источники данных, похожие проекты и стек.
- Создан статический MVP без внешних зависимостей.
- Добавлены учебные таблицы: `students`, `courses`, `enrollments`.
- Добавлены задания по темам `SELECT`, `WHERE`, `ORDER BY`, `JOIN`, `GROUP BY`.
- Реализован SQL-редактор.
- Реализован запуск ограниченного набора безопасных учебных SQL-запросов.
- Реализована автоматическая проверка результата.
- Реализован прогресс прохождения.
- Реализованы мок-подсказки ИИ-помощника.
- Добавлен Python backend.
- Добавлена интеграция с локальным PostgreSQL.
- Добавлен endpoint `/api/query` для выполнения безопасных `SELECT`-запросов.
- Добавлен endpoint `/api/hint` для LLM-подсказок или мок-режима без API-ключа.
- Добавлен запуск через `start_backend.bat`.

## Что реализовано частично

- ИИ-помощник работает в мок-режиме, пока не задан `LLM_API_KEY`.
- PostgreSQL запускается локально на порту `55432`, чтобы не конфликтовать со стандартным портом `5432`.

## Что планируется дальше

- Улучшить backend до FastAPI.
- Добавить Docker Compose для простого запуска.
- Подключить настоящий LLM API для генерации подсказок и объяснения ошибок.
- Добавить новые наборы заданий и уровни сложности.

## Риски

- Без backend нельзя полностью показать реальную изоляцию SQL-песочницы.
- Для настоящего ИИ-помощника понадобится API-ключ или доступ к OpenAI-compatible сервису.
- Для полноценной версии нужно будет установить runtime для backend/frontend разработки.
